## 260409 구현 기록

### Test 01 (기존 이미지)

#### Depth estimation

![Depth Color A](./01/depth_color.png)
![Depth Objects Pointcloud A](./01/depth_objects_pointcloud.png)

- `DepthPro(apple/DepthPro-hf)`로 픽셀별 absolute depth를 추정한다.
- 기존 relative depth는 장면마다 깊이 스케일이 달라 객체 간 실제 거리 비교가 불안정했다.
- absolute depth를 사용하면 카메라 추정값과 직접 결합해 거리/크기 해석 일관성을 높일 수 있다.
- depth 결과를 빠르게 확인하기 위해 이미지 중심 기반 간이 투영으로 점을 만든다.

$X = \frac{u-c_x}{W}Z,\qquad Y = -\frac{v-c_y}{H}Z\frac{H}{W},\qquad Z = \max(D(u,v), 10^{-3})$
- $W, H$: 이미지 전체 크기
- $c_x = \frac{W}{2}$, $c_y = \frac{H}{2}$
- $u,v$: 픽셀 좌표
- $D(u,v)$: depth map의 픽셀 깊이값

> $X,Y$ 스케일을 맞추기 위해 가로/세로를 사실상 `W` 기준으로 정규화한다 (`Y` 식의 `H/W` 항).

> 이미지 좌표계는 아래로 갈수록 값이 커지므로, 3D 좌표계의 위쪽(+Y)과 맞추기 위해 `Y`에 음수 부호를 둔다.

> depth가 0에 가까울 때 수치 불안정을 막기 위해 $Z=\max(D(u,v),10^{-3})$ 하한을 사용한다.

#### Camera calibration

![Camera Calibration Pointcloud A](./01/camera_depth_pointcloud.png)

- `PerspectiveFields`로 단일 이미지에서 `vfov`, `pitch`, `roll`, `rel_cx`, `rel_cy` 등을 추정한다.
- `핀홀 카메라 모델` 가정에 따라 `fx, fy, cx, cy, horizon`을 복원한다.

$f_y = \frac{H/2}{\tan(vfov/2)},\qquad f_x=f_y$

$c_x = W(0.5+rel\_{c_x}),\qquad c_y = H(0.5+rel\_{c_y})$

$horizon\_y = c_y - f_y \tan(pitch)$

- 카메라 왜곡이 보정된 3D 좌표를 계산하여 시각화한다.

$X_c = \frac{u-c_x}{f_x}Z,\qquad Y_c = -\frac{v-c_y}{f_y}Z,\qquad Z_c = \max(D(u,v)\cdot s, 10^{-3})$

- 이후 `pitch/roll` 회전을 적용해 카메라 원점 좌표계에서 월드 좌표계로 변환하여 저장한다.

$dx = u-c_x,\ dy = v-c_y$

$dx' = dx\cos(-r)-dy\sin(-r),\qquad dy' = dx\sin(-r)+dy\cos(-r)$

$X = \frac{dx'}{f_x}Z,\qquad Y = \left(-\frac{dy'}{f_y}-\tan(p)\right)Z,\qquad Z = Z$
- $r$: roll, $p$: pitch

#### Layout preview

![Layout Preview A](./01/layout_preview.png)

- 기존 2D bbox 중심 계산 방식으로 배치가 잘 되지않아 객체별 3D 점군에서 직접 크기/위치를 계산하는 방식으로 수정했다.
- 각 축에 대해 퍼센타일 경계(5~95%)를 사용해 outlier를 줄인 축정렬 박스(cuboid)를 만든다.
- 점군 기반 경계 추정으로 객체별 위치/크기를 직접 계산해 초기 배치의 기하적 일관성을 높인다.

$x_{min},x_{max}=Q_{5},Q_{95}\ (X),\quad y_{min},y_{max}=Q_{5},Q_{95}\ (Y),\quad z_{min},z_{max}=Q_{5},Q_{95}\ (Z)$

$center=\left(\frac{x_{min}+x_{max}}{2},\frac{y_{min}+y_{max}}{2},\frac{z_{min}+z_{max}}{2}\right)$

$size=\left(x_{max}-x_{min},\ y_{max}-y_{min},\ z_{max}-z_{min}\right)$
